1.	Se connecter en Local a la RPI en tapant dans l'URL : IPRPI:8080
	L'ID et Administrator et le MDP est celui rentrer pendant la configuration de l'installation de IOT

2.	Se rendre dans la section "Apps" puis "Apps Manager"

3.	Cliquer sur "Choisir un Fichier"

3.1	Si ce n'est pas déjà fait, installer les fichiers présent dans le fichier "A Installer avant.."

4.	Ensuite cliquer sur "Choisir un Fichier" et selectionner le fichier ".appxbundle"

5.	Pour finir il ne reste plus qu'a lancer l'application
